# Chat bot Telegram
## 1. Ligando o servidor
1.1 Considerando que ja tenha o pip instalado,instale o modulo de virtualenv.
```
pip3 install virtualenv 
```
1.2 No terminal e localizado na pasta do projeto.
```
source bin/activate
```
1.3 Após isso ligue o servidor
```
python3 src/main.py 
```
